1. Change password:
`passwd`
2. change hostname:
`sudo nmtui`
3. Connect to wifi:
`nmtui`
4. Update packages:
`sudo apt-get update`
5. Install git:
`sudo apt-get install git`
6. get this repo:
`git clone https://github.com/apare/pocketchip.git`
7. Go inside pocketchip/:
`cd pocketchip`
8. make executable install.sh:
`sudo chmod +x install.sh`
9. run install.sh: `sudo ./install.sh`
